From Assignment-2 folder do all following to get graphs

Setup conda environment using "requirements.yml"

Run following commands:

conda env create --name envname --file=environments.yml
conda activate envname


To run PIMA Indian Diabetes dataset:

For NN Part:

Run "python diabetes.py", it will generate graphs in "figures"

For KnapSack Part:

Run "python knapSack.py", it will generate graphs in "figures"

For Continuous Peaks Part:

Run "python continuousPeaks.py", it will generate graphs in "figures"

For One Max Part:

Run "python oneMax.py", it will generate graphs in "figures"
